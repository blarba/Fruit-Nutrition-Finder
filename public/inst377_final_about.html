This is just a simple website utilizing the Fruityvice API and some of its fetch calls to gather information on fruits of your choosing. It uses several calls, which are:

  • api/fruit/{name} - searches for the fruit name
  • /api/fruit/family/{family name} - searches for all fruits with family name
  • /api/fruit/genus/{genus} - searches for all genus of same name
  • /api/fruit/order/{order} - searches for all fruits within the given order
  • /api/fruit/{nutrition}?min=0&max=1000 - searches for a nutrition category and filters
  • /api/fruit/all - gets all fruits in the database
